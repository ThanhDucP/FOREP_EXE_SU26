# Kế Hoạch Triển Khai MVP FOREP EXE

Mục tiêu: đưa sản phẩm nhỏ, nhanh, dễ dùng cho chủ doanh nghiệp có 2-30 nhân viên.

## Ưu tiên

1. Workspace registration.
2. OWNER/EMPLOYEE auth.
3. Employee list và create employee.
4. Task list, create task, task detail.
5. Progress update.
6. Workload dashboard.
7. AI recommend assignee.
8. Daily report.
9. Notifications.

Không làm trong MVP:

- Phân cấp tổ chức.
- Team management.
- Phòng ban.
- Vai trò quản lý trung gian.
- Sprint/kanban/project management.
- Voice.

## Milestone 1: Core CRUD

- Register workspace.
- Login.
- Create employee.
- Create task.
- Assign task.
- Update progress.
- Complete task.

## Milestone 2: Visibility

- Owner dashboard.
- Employee dashboard.
- Workload dashboard.
- Overdue task list.
- Missing update list.

## Milestone 3: AI

- Recommend assignee.
- Workload summary.
- Business summary.

## Milestone 4: Reporting

- Daily report.
- Review report.
- Notifications.
